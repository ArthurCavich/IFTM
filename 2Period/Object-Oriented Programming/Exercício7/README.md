Exercício 7 - Métodos Estáticos

Arquivos:
- App.java: contém o método main, le() e exibe().
- Triangulo.java: classe que representa o triângulo, com construtor, getters/setters e tipoTriangulo().

Como compilar e executar (Windows PowerShell):

1. Abra o PowerShell e navegue até a pasta do projeto:

```powershell
cd "c:\Users\arthu\OneDrive\Documentos\GitHub\IFTM\2Period\Object-Oriented Programming\Exercício7"
```

2. Compile:

```powershell
javac App.java Triangulo.java
```

3. Execute:

```powershell
java App
```

Insira os três lados quando solicitado. O programa verifica se formam triângulo e exibe o tipo: "equilátero", "isósceles" ou "escaleno".
